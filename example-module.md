\## Installation



\### Install via Nilsoft Launcher (Recommended)

1\. Open the Nilsoft Launcher  

2\. Navigate to \*\*Modules\*\*  

3\. Select \*\*Example Module\*\*  

4\. Click \*\*Install\*\*  



The launcher will:

\- download the module  

\- verify the SHA256 hash  

\- install it into the correct Nil‑AOS directory  

\- register it with the system
