\# Example Bundle  

A curated set of modules and agents for new Operators.

\## Overview

The Example Bundle provides a unified, doctrine‑aligned set of Nilsoft modules and agents.  

It is designed to give Operators a clean, immediate capability upgrade with zero drift and minimal configuration.

\## Installation



\### Install via Nilsoft Launcher (Recommended)

1\. Open the Nilsoft Launcher  

2\. Navigate to \*\*Bundles\*\*  

3\. Select \*\*Example Bundle\*\*  

4\. Click \*\*Install\*\*  



The launcher will:

\- download all components  

\- verify each SHA256 hash  

\- install modules and agents into the correct directories  

\- register everything with Nil‑AOS
